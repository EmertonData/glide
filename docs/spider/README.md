# Spider Text-to-SQL with LLM-Judge Labels

This dataset extends [Spider 1.0](https://yale-lily.github.io/spider) with SQL predictions from Claude Haiku and two correctness labels per example: a programmatic human label (SQL execution comparison) and an LLM judge label (Claude Sonnet).

## Files

| File | Description |
|------|-------------|
| `spider_dataset.parquet` | Full dataset with predictions and labels |
| `scripts/` | Reproduction scripts (see below) |

## Dataset statistics

- **Source**: Spider 1.0 training split (`train_spider.json`)
- **Databases**: subset of the 200 Spider databases, selected by running `scripts/explore_dataset.py`
- **Labels**: binary (0 = incorrect, 1 = correct)

## Schema

| Column | Type | Description |
|--------|------|-------------|
| `example_id` | string | Unique identifier (`spider_train_{original_index}`) |
| `db_id` | string | Database name |
| `question` | string | Natural language question |
| `gold_sql` | string | Reference SQL query from the Spider benchmark |
| `predicted_sql` | string | SQL generated by Claude Haiku (`claude-haiku-4-5-20251001`) |
| `llm_judge_label` | int | 1 if Claude Sonnet (`claude-sonnet-4-6`) judges the prediction correct, 0 otherwise |
| `human_label` | int | 1 if executing both SQL queries against the SQLite database yields identical sorted result sets, 0 otherwise |

Both prompts include the full database schema (tables, columns, types, primary keys, foreign keys) parsed from `tables.json`.

## Reproduction

Install dependencies:

```bash
pip install -r scripts/requirements.txt
```

Then run the scripts in order from the repository root:

```bash
# 1. Download and extract Spider 1.0
python scripts/download_dataset.py

# 2. Inspect per-database statistics to inform database selection
python scripts/explore_dataset.py

# 3. Generate SQL predictions with Claude Haiku
#    (set N_DATABASES and N_PER_DATABASE in the script first)
python scripts/generate_predictions.py

# 4. Generate LLM judge labels and human labels
python scripts/generate_llm_judge_labels.py
python scripts/generate_human_labels.py

# 5. Merge and push to HuggingFace
python scripts/build_dataset.py
```

Steps 3 and 4 write checkpointed JSONL files under `data/` and can be safely interrupted and resumed. An `ANTHROPIC_API_KEY` environment variable must be set before running them.
