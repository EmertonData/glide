What is GLIDE?

GLIDE (Generated Label Inference & Debiasing Engine) is a Python library for rigorous evaluation of GenAI systems.

  • LLM judges are biased. Proxy evaluators systematically over- or under-estimate true performance.
  • Human annotation is expensive. Full labeling at scale is rarely feasible.
  • GLIDE bridges the gap. Using prediction-powered inference, it combines a small human-labeled sample with your existing proxy scores to produce valid, debiased performance estimates.
Key features
  • Bias-corrected performance estimates from a handful of human labels and your existing LLM judge scores
  • Smart annotation budget allocation to minimise the human labeling needed for reliable results
  • Guaranteed confidence intervals regardless of proxy quality
  • Compatible with any LLM judge or human evaluation pipeline
