What is GLIDE?

Teams deploy GenAI systems with unknown true performance, forced to choose between scalable-but-biased LLM judges and unaffordable human annotation. GLIDE takes a third path: a small human-labeled sample characterises the bias in your proxy scores, then Prediction-Powered Inference corrects it. The result is a certified performance estimate with defensible confidence intervals, at a fraction of the annotation cost.

Key features
  • Prediction-Powered Inference (PPI) and Assumption-free Statistical Inference (ASI) estimators
  • Stratified sampling for subgroup-level evaluation
  • Cost-optimal annotation budget allocation
  • Classical estimators as a baseline, in the same unified API
  • Guaranteed confidence intervals regardless of proxy quality
  • Compatible with any LLM judge or human evaluation pipeline
